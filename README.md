# Gallery Project Made by Asma

This is a React + Vite based gallery project.

## 🚀 Features
- Image gallery UI
- Responsive design
- Fast performance using Vite

## 🛠️ Tech Stack
- React
- Vite
- JavaScript
- CSS

## 📂 Setup Instructions
1. Clone the repository
2. Run `npm install`
3. Run `npm run dev`
